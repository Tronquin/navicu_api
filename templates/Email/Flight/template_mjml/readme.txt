# MJML

MJML es un framework que incluye una interfaz de línea de comandos, una biblioteca Node.js y una página try-it-live en su sitio web oficial.

## Instalación

Usar el paquete [mjml](https://mjml.io/documentation) e instalar.

```bash
npm install mjml
```

## Compilación 

```python
Crear archivo con extencion .mjml

# Compilar template.mjml a output.html
mjml template.mjml -o output.html

# Ver template.mjml y auto-compilar cada cambio
mjml -w template.mjml -o index.html
```
